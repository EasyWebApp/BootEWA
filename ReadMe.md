## 特性优势

 - 迁移平滑：兼容已有的 jQuery、BootStrap 架构，代码修改集中在 **精简、拆分、规范**

 - 学习容易：EasyWebApp 引擎本身没有发明 **HTML 5、CSS 3、JavaScript** (ES 5+) 之外的新语法

 - 使用简单：依赖库、组件均 **异步加载**，无需任何编译、打包、构建工具链
